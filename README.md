
Para ejecutar estos archivos hay que instalar el jupyter notebook especificado en requirements. Se puede hacer en la consola del cmd insertando el comando pip install jupyter[lab]. 
Una vez que esté instalado hay que ejecutarlo (cosa que también se puede hacer desde consola poniendo el comando jupyter lab). Antes de empezar a probar el programa hay que asegurarse de que esté el archivo notebook (tipo ipynb), pero también el módulo formato python con las sentencias y definiciones (llamado modulo_funciones.py).
